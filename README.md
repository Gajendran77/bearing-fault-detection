# Bearing Fault Detection using Machine Learning

This project detects faults in rolling element bearings using vibration signal analysis.

## Features
- Machine learning based fault classification
- Random Forest model
- Web interface built with Streamlit
- Upload vibration CSV data to detect bearing faults

## Dataset
Case Western Reserve University Bearing Dataset.

## Web App
Upload vibration data to automatically detect:
- Normal bearing
- Inner race fault
- Ball fault
- Outer race fault
